## Project 1 — Portfolio & Investment Analysis

**Folder:** `Project1-Portfolio`

### Overview
Comparative portfolio and investment analysis of two publicly traded companies:
- **LVMH Moet Hennessy Louis Vuitton SE (MC.PA)** — Euronext Paris
- **Tapestry, Inc. (TPR)** — NYSE

Weekly adjusted closing price data was sourced from Yahoo Finance covering March 2025 to March 2026 (53 weekly observations).

### Methods Applied
- Data acquisition and preparation from Yahoo Finance
- Weekly return calculation: R(t) = (P(t) − P(t-1)) / P(t-1)
- Descriptive statistics: mean return, standard deviation, variance, min/max
- Sharpe Ratio calculation (risk-free rate: 4.5% p.a.)
- Pearson correlation coefficient for diversification analysis
- 50/50 equally-weighted portfolio construction
- Data visualization: comparative line chart of adjusted close prices

### Key Results
